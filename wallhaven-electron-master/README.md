<h1 align="center">wallhaven-electron</h1>

<div align="center">
一款基于 Electron 壁纸客户端 。
</div>

- gitee：https://gitee.com/ml13/wallhaven-electron
- github：https://github.com/MaLuns/wallhaven-electron

## 运行
```
# 安装依赖
yarn

# 运行开发模式 （注意这里需要两个都跑起来)
npm run dev:web
npm run dev:exe

# 打包安装文件 
npm run build:web
npm run build:exe

```

## 项目结构
```
- build 打包图标
- public 静态资源
- layout 前端文件
- src
  - main 主线程目录
  - renderer 渲染线程（前端打包输出目录）
```
